CV Web - Projet HTML/CSS/JS

Description

Ce projet consiste en la réalisation d'un CV interactif et responsive en HTML, CSS et JS. Il présente les informations personnelles ainsi que le profil, le parcours scolaire, les projets académiques, les compétences techniques et transversales, les langues parlées et les cemtre d'interêt et pour finir un formulaire de contact destiné au recruteur puis en bas les liens personnelles vers les réseaux sociaux.

Technologies utilisées

- HTML5
- CSS3
- JavaScript (pour le menu mobile, les skills animations et le mode sombre/claire)

Instructions pour ouvrir le projet

1. Télécharger ou cloner le projet.
2. Scanner le QR code.
3. Double-cliquer sur le fichier "index.html" ou l'ouvrir avec un navigateur web (Google Chrome,    Microsoft Edge, Firefox, etc.).
4. Pour le développement, il est recommandé d'utiliser Visual Studio Code avec l'extension Live Server afin d'afficher le projet localement.
